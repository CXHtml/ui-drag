参数类型默认值数据流向描述
dataobject绑定属性
data.proxystring
Dragable.Proxy
Element
function		'self'outer @=> inner拖拽代理，即拖拽时移动的元素。默认值为`clone`，拖拽时拖起自身的一个拷贝；当值为`self`，拖拽时直接拖起自身。也可以用`<draggable.proxy>`自定义代理，或直接传入一个元素或函数。其他值表示不使用拖拽代理。
data.axisstring'both'outer => inner拖拽代理移动时限制的轴向，`both`表示可以在任意方向上移动，`horizontal`表示限制在水平方向上移动，`vertical`表示限制在垂直方向上移动
data.disabledbooleanfalseouter => inner是否禁用
data.classstring'z-draggable'outer => inner可拖拽时（即disabled=false）给该元素附加此class
data.sourceClassstring'z-dragSource'outer => inner拖拽时给起始元素附加此class
data.proxyClassstring'z-dragProxy'outer => inner拖拽时给代理元素附加此class
